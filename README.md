## 如何学习 Python

### 1. 搭建环境

- 安装 **Python 3.10+**（推荐 3.13）。
- 使用 **PyCharm** 或 VS Code 作为 IDE。
- 学习过程中建议为每个项目创建独立的 **虚拟环境**（`python -m venv venv`），避免包冲突。

### 2. 分阶段学习路线

#### 阶段一：Python 核心基础（第 1–3 章）
- 理解计算机组成、编程语言分类（编译型 / 解释型）。
- 掌握 **字面量、变量、标识符、常量、注释、字符编码**。
- 熟悉 **数据类型**（int、float、str）、类型转换、运算符（算术、赋值、比较、逻辑、成员）。
- 学会使用 `input()` 获取用户输入，`print()` 格式化输出。

#### 阶段二：流程控制与函数（第 4–5 章）
- **分支语句**：`if-elif-else`。
- **循环**：`while`、`for`，以及 `break`、`continue`、嵌套循环。
- **函数**：定义、参数（位置、关键字、默认值、可变参数）、返回值、多种作用域（`global`、`nonlocal`）。
- 递归、函数说明文档、综合练习。

#### 阶段三：数据容器（第 6 章）
- **列表**：增删改查、常用方法（`append`、`pop`、`sort` 等）。
- **元组**：不可变序列、解包。
- **字符串**：不可变，常用方法（`split`、`replace`、`strip`）。
- **集合**：无序不重复，数学运算（并集、交集等）。
- **字典**：键值对，遍历、常见方法（`keys`、`values`、`items`）。
- 通用操作：`len`、`sorted`、`in`、类型互转。

#### 阶段四：面向对象（第 7 章）
- **类与对象**：`class`、`__init__`、实例方法。
- **继承**：单继承、多重继承、`super()`、方法重写。
- **属性权限**：公有、受保护（`_`）、私有（`__`），`@property` 装饰器。
- **魔法方法**：`__str__`、`__len__`、`__lt__` 等。
- **多态**：标准多态与鸭子类型。
- **抽象类**：`ABC` 与 `abstractmethod`。
- 综合案例：学生成绩管理系统。

#### 阶段五：函数进阶与常用内置工具（第 8 章）
- 函数是一等对象，可作为参数或返回值。
- 匿名函数 `lambda`、`map`、`filter`、`sorted`、`reduce`。
- 列表 / 字典推导式。
- 浅拷贝与深拷贝（`copy` 模块）。
- 作用域 LEGB 规则。
- 闭包：记住状态的内部函数。
- 装饰器：函数装饰器与类装饰器，带参数装饰器。
- 类型注解（`str`、`list[int]`、`-> int` 等）。

#### 阶段六：错误处理与模块化（第 9–10 章）
- 异常体系：`try-except-else-finally`，`raise` 自定义异常。
- **模块**：`import`、`from…import`、`__name__ == '__main__'`。
- **包**：包含 `__init__.py` 的文件夹，管理多模块。
- 标准库初探：`os`、`random`、`time`、`math` 等。
- 第三方包：`pip` 安装、虚拟环境使用。

#### 阶段七：迭代器、生成器与文件操作（第 11–12 章）
- **可迭代对象**与**迭代器**：`__iter__`、`__next__`。
- **生成器**：`yield` 关键字，惰性计算，节省内存。
- 文件读写：`open()`，`with` 上下文管理器，`read`、`write`、`readlines`。
- 目录操作：`os.path`、`shutil.rmtree` 等。

#### 阶段八：并发编程（第 13–14 章）
- 进程与线程概念、GIL 全局解释器锁。
- `multiprocessing.Process`、`Thread`、对应的池 `Executor`。
- 锁（`Lock`、`RLock`）、队列（`Queue`）、管道（`Pipe`）。
- **协程**：`async/await`、`asyncio.run()`、`create_task()`、`gather()`。
- 实战：使用 `aiohttp` 异步下载图片。

## 注意事项

- 所有代码均基于 **Python 3.10+** 编写，部分特性（如 `|` 联合类型）需要 3.10 以上。
- 多进程部分在 Windows 下必须在 `if __name__ == '__main__':` 保护中运行。
- 异步代码需要在 Python 的异步环境中执行（直接运行 `asyncio.run(main())` 即可）。
